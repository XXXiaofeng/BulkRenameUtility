# Docker Desktop 启动指南

## ✅ 安装完成

Docker Desktop 已成功安装到您的系统中。

## 🚀 下一步：手动启动 Docker

### 方法1：通过 Launchpad 启动
1. 打开 Launchpad（按 F4 或点击 Dock 中的火箭图标）
2. 找到 Docker 图标（鲸鱼图标）
3. 点击启动 Docker

### 方法2：通过 Spotlight 启动
1. 按 `Command + Space` 打开 Spotlight
2. 输入 "Docker"
3. 按回车启动

### 方法3：通过应用程序文件夹
1. 打开 Finder
2. 进入 "应用程序" 文件夹
3. 双击 Docker.app

## ⏱️ 等待启动完成

首次启动 Docker 需要一些时间，请耐心等待：
- 菜单栏会出现鲸鱼图标 🐳
- 图标动画停止表示启动完成
- 可能会要求输入系统密码授权

## 🔍 验证启动成功

当 Docker 完全启动后，在终端中运行：
```bash
docker --version
```

如果显示版本号，说明 Docker 已成功启动！
